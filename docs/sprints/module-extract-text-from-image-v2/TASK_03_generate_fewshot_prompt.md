## Goal / value

Create a prompt generation script that builds a `ChatPromptTemplate` with 3 few-shot examples (synthetic images + expected tool call outputs), serializes it with `dumpd()`, and writes `nl.json`. The few-shot examples teach the model how to use the `ExtractedText` tool correctly for Dutch text extraction, mixed-language filtering, and empty-result handling.

## Implementation steps (developer-facing)

### Step 1: Create `nl_processing/extract_text_from_image/prompts/generate_nl_prompt.py`

Create a new script file with the following structure:

```python
"""Generate the Dutch extraction prompt (nl.json) with few-shot examples.

Usage:
    uv run python nl_processing/extract_text_from_image/prompts/generate_nl_prompt.py

This script:
1. Generates 3 synthetic test images using benchmark.generate_test_image()
2. Encodes them to base64
3. Builds a ChatPromptTemplate with few-shot examples (HumanMessage + AIMessage pairs)
4. Serializes with dumpd() and saves to nl.json
"""

import json
import tempfile

from langchain_core.load import dumpd
from langchain_core.messages import AIMessage, HumanMessage, SystemMessage
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

from nl_processing.extract_text_from_image.benchmark import generate_test_image
from nl_processing.extract_text_from_image.image_encoding import encode_path_to_base64
```

### Step 2: Define the 3 few-shot examples

```python
SYSTEM_INSTRUCTION = (
    "Je bent een tekst-extractie assistent. "
    "Extraheer alleen de Nederlandse tekst uit het aangeboden beeld. "
    "Behoud de originele documentstructuur als markdown "
    "(koppen, nadruk, regelafbrekingen). "
    "Negeer tekst in andere talen. "
    "Retourneer alleen de geëxtraheerde tekst, zonder commentaar of uitleg."
)

# Example 1: Normal Dutch text
EXAMPLE_1_TEXT = "De kat zit op de mat"
EXAMPLE_1_EXPECTED = "De kat zit op de mat"

# Example 2: Mixed Dutch + Russian (Russian renders as garbled chars in cv2)
EXAMPLE_2_TEXT = "Welkom bij ons\nДобро пожаловать"
EXAMPLE_2_EXPECTED = "Welkom bij ons"

# Example 3: English-only text (no Dutch)
EXAMPLE_3_TEXT = "The quick brown fox jumps over the lazy dog"
EXAMPLE_3_EXPECTED = ""
```

### Step 3: Build the prompt

```python
def _generate_image_b64(text: str, width: int = 800, height: int = 200) -> str:
    """Generate a synthetic image and return its base64 encoding."""
    with tempfile.NamedTemporaryFile(suffix=".png", delete=False) as tmp:
        generate_test_image(text, tmp.name, width=width, height=height, font_scale=1.2)
        b64, media_type = encode_path_to_base64(tmp.name)
    return f"data:{media_type};base64,{b64}"


def _make_example_human(image_data_url: str) -> HumanMessage:
    """Create a HumanMessage with an image content block."""
    return HumanMessage(content=[
        {"type": "image_url", "image_url": {"url": image_data_url}},
    ])


def _make_example_ai(expected_text: str, call_id: str) -> AIMessage:
    """Create an AIMessage with a tool_call for ExtractedText."""
    return AIMessage(
        content="",
        tool_calls=[{"name": "ExtractedText", "args": {"text": expected_text}, "id": call_id}],
    )


def build_prompt() -> ChatPromptTemplate:
    """Build the Dutch extraction prompt with 3 few-shot examples."""
    img1 = _generate_image_b64(EXAMPLE_1_TEXT)
    img2 = _generate_image_b64(EXAMPLE_2_TEXT, height=200)
    img3 = _generate_image_b64(EXAMPLE_3_TEXT)

    return ChatPromptTemplate.from_messages([
        SystemMessage(content=SYSTEM_INSTRUCTION),
        _make_example_human(img1),
        _make_example_ai(EXAMPLE_1_EXPECTED, "call_example_1"),
        _make_example_human(img2),
        _make_example_ai(EXAMPLE_2_EXPECTED, "call_example_2"),
        _make_example_human(img3),
        _make_example_ai(EXAMPLE_3_EXPECTED, "call_example_3"),
        MessagesPlaceholder(variable_name="images"),
    ])
```

### Step 4: Main block to generate and save

```python
OUTPUT_PATH = "nl_processing/extract_text_from_image/prompts/nl.json"


if __name__ == "__main__":
    prompt = build_prompt()
    data = dumpd(prompt)
    with open(OUTPUT_PATH, "w", encoding="utf-8") as f:
        json.dump(data, f, indent=2)
    print(f"Prompt saved to {OUTPUT_PATH}")  # noqa: T201 — dev script
    print(f"Messages: {len(prompt.messages)}")  # noqa: T201 — dev script
    print(f"Input variables: {prompt.input_variables}")  # noqa: T201 — dev script
```

## Edge cases

- `generate_test_image()` uses `cv2.FONT_HERSHEY_SIMPLEX` which does NOT support Cyrillic. Russian text in Example 2 will render as `?` or boxes. This is intentional — the model should still recognize it as non-Dutch and ignore it. The few-shot example teaches the model the correct behavior: "when you see non-Dutch characters in the image, exclude them from the output."
- `tempfile.NamedTemporaryFile` with `delete=False` creates temp files that are not cleaned up. Use `tempfile.TemporaryDirectory` as context manager instead for cleaner cleanup, or just use `tempfile.mkstemp`. The script runs once for generation — temp files are negligible.

## Notes / risks

- **Risk**: The base64 images in `nl.json` make the file large and the prompt token-heavy.
  - **Mitigation**: Synthetic images are small (800×200 white background with black text). Base64 encoding of a simple PNG is ~2-5KB per image. Total prompt size with 3 images is manageable for GPT-4 vision (128K context).
- **Risk**: The `generate_nl_prompt.py` script must be re-run if example text or image parameters change.
  - **Mitigation**: This is by design — the script is the source of truth, `nl.json` is the generated artifact. Document this in the script docstring.
- **Risk**: `tempfile.NamedTemporaryFile` behavior differs on Windows (cannot read while open).
  - **Mitigation**: Use `delete=False` and clean up manually, or use `tempfile.TemporaryDirectory()`. The project runs on macOS (verified from environment).
